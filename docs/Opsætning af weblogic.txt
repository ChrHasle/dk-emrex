Opsætning EMREX weblogic server
-------------------------------
1) Weblogic installation
	Sæt JDK i PATH
	Placer dig i sti hvor installation skal ske
	java -jar 'installationfil'.jar
	Installation er nu i ./wls12212
	Skift til wls12212
	Udfør wlserver\server\bin\setWLSEnv.cmd
	mkdir emrex
	cd emrex
	%JAVA_HOME%\bin\java.exe  weblogic.Server
		User : weblogic
		Password : weblogic123
	Restart weblogic
	Peg browser på http://localhost:7001/console/

2) SMP og NCP installation
	Sæt spring boot profile til 'test' i startWebLogic.cmd : -Dspring.profiles.active=test

	Generer valideret med zerossl.com eller self-signed med http://www.selfsignedcertificate.com/
	
	
4) Apache opsætning
	Installer apache 2.4
	Installer weblogic plugins i Apache
