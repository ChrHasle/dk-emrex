Opsætning EMREX weblogic server
-------------------------------
1) Weblogic installation
	Sæt JDK i PATH
	Placer dig i sti hvor installation skal ske
	java -jar fmw_12.1.3.0.0_wls.jar.jar
	Installation er nu i ./wls12212
	Skift til wls12212
	Udfør wlserver\server\bin\setWLSEnv.cmd
	mkdir emrex
	cd emrex
	%JAVA_HOME%\bin\java.exe  weblogic.Server
		User : weblogic
		Password : Welcome1
	Restart weblogic
	Peg browser på http://localhost:7001/console/
