OBS! Udskift ${env} nedenfor med dev, test eller prod for ønsket miljø.

Kopier stads-registry-${env}.json til /etc/emrex

Kopier war file til ønsket placering.

Start applikation på kommandolinie med 
	java -jar -Dspring.profiles.active=${env} ncp-0.0.2-SNAPSHOT.war

Logning sker til STDOUT.	

Beskrivelse af init.d opsætning : http://docs.spring.io/spring-boot/docs/current/reference/html/deployment-install.html
