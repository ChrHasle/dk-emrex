Apache server 
	Installer Apache 2.2 på dans-emrexws.kmd.dk
	Konfigurer med httpd.test.conf fra NCP projekt lagt som etc/httpd/conf/httpd.conf samt mime.types
	Start server med sudo /sbin/service httpd start
	Restart server med sudo /sbin/service httpd restart
	Stop server med sudo /sbin/service httpd stop
